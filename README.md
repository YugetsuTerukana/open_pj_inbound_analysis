# インバウンド市場分析プロジェクト  
## 観光 × 消費 × 行動データによる市場構造の整理

---

## Overview

本プロジェクトは、訪日外国人旅行者に関する公開統計データを用いて、日本のインバウンド市場を **「市場規模（量）」「消費構造（質）」「旅行中の行動」「訪問地」** という複数の視点から整理・分析するデータ分析プロジェクトである。

従来のインバウンド分析では、訪日客数や消費総額といった単一指標に基づく評価が中心となりがちであった。  
しかし、それらの指標だけでは、

- なぜ特定の市場で消費水準が高いのか  
- 行動の違いがどのような形で現れているのか  
- 市場ごとにどのような特性差があるのか  

といった、施策検討や市場理解に必要な論点を十分に整理することは難しい。

本プロジェクトでは、 **分析自体を目的化するのではなく、市場をどのような構造として捉え直せるかを整理すること** を重視し、以下の設計思想に基づいて分析を行っている。

- 各 Notebook は明確な分析テーマを持ち、内容が過度に重複しないこと  
- 消費・行動・訪問地といった異なるデータを、後から並べて解釈できる構造とすること  
- 国・地域ごとの市場特性を、多面的に説明できる材料を提供すること  

本プロジェクトでは、Notebook による分析設計・検証と、Streamlit による可視化・共有を分離して管理している。  
**Streamlitによる可視化・共有を起点として設計された分析テーマについて、Notebook側では評価軸・指標の整理や分析手順、結果解釈の検証を行っている。**  
両者は同一の分析思想に基づき、相互に補完する関係にある。

その結果、本プロジェクトは単なる可視化集ではなく、**インバウンド市場を量・質・行動・空間の観点から整理し、市場理解や施策検討の前提条件を明確にするための分析基盤**として位置づけられる。

---

## プロジェクト構成と役割分担

本プロジェクトは、以下の2つのレイヤーで構成されている。

### 分析レイヤー（Notebook）
- 分析テーマごとの評価軸・指標の整理  
- 分析手順および結果解釈の整理  
- 各分析結果を読み解くための前提条件の明示  

### 可視化・共有レイヤー（Streamlit）
- 分析テーマの可視化・プロトタイピング  
- 非分析者（自治体・DMO 等）への共有  
- ダッシュボードとしての再利用  

Notebook と Streamlit は **分析テーマ単位で対応**しており、  
Streamlit で提示される指標や構造について、  
Notebook 側でその妥当性や解釈を検証・整理する役割分担として設計されている。

## 使用データと前処理（分析基盤の前提）

本プロジェクトでは、JNTOおよび観光庁が公開する複数の公的統計データを使用している。  
これらの統計は、調査目的や集計方法が異なるため、期間区分、国・地域表記、項目構成、データ形式が統一されていない。

そのため、分析および可視化を行う前段として、以下の方針に基づき前処理を行っている。

- 複数年・複数ファイルに分かれた統計データを結合し、年・四半期といった時間軸で扱える形式へ整理  
- 国・地域名や項目名称の表記ゆれを整理し、比較単位となる国・地域の定義を統一  
- クロス集計形式の表を、集計・可視化に適したデータ構造へ変換  
- 数値項目（訪日客数、消費単価、実施率など）について、分析可能な数値型として扱えるよう整形  

前処理後のデータは 配下に格納されており、各NotebookおよびStreamlitアプリでは、これらの加工済みデータを共通の分析基盤として参照している。

前処理後のデータは、ローカル環境において「data/」 配下に配置されることを前提としており、各 Notebook および Streamlit アプリでは、これらの加工済みデータを共通の分析基盤として参照する設計としている。

なお、本リポジトリにはデータファイルそのものは同梱しておらず、データの再配布は行っていない。

## Notebook一覧と分析内容

### 001_Overview.ipynb  
分析全体の前提条件、使用データ、分析設計の考え方を整理。

### 010_Market_Potential.ipynb  
**訪日客数（量）× 1人あたり消費額（質）による市場ポテンシャル分析**

- 市場規模と消費水準を分けて評価  
- 国・地域別の市場ポジションを整理  

### 020_Consumption_Structure.ipynb  
**消費単価の差を費目構造から分解**

- 高単価市場ではサービス消費の比重が高い  
- 消費単価の差が構造的要因によるものであることを確認  

### 031_Behavior_PCA.ipynb  
**旅行中行動データの主成分分析**

- 行動を複数の志向軸（PC1〜PC3）として構造化  
- 特に PC3（自然・地方・体験志向）を重要軸として抽出  

### 032_Behavior_Timeseries.ipynb  
**主成分スコアの時系列推移分析**

- 国別に行動志向が変化していることを確認  
- 一部市場で自然・地方志向の高まりを確認  

### 033_Destination_VisitRate.ipynb  
**訪問率データによる訪問地選択の変化分析**

- 地方・自然資源を有する地域で訪問率が伸長

## Data policy

本リポジトリは分析設計および実装例を示すポートフォリオを目的としており、JNTO・観光庁の公開統計データを利用した分析を想定しています。  
ただし、データファイルそのものは再配布せず、取得方法および分析手順のみを公開しています。
