# Data Policy / データの取り扱いについて

本プロジェクトでは、JNTOおよび観光庁等が公開している公的統計データを前提として、インバウンド市場分析の設計および検証を行っています。

## 使用データの出典

本プロジェクトで想定している主なデータソースは以下の通りです。

- JNTO「観光旅行期間中の行動内容」
- JNTO「訪日外客統計」
- JNTO「訪日外客訪問先」
- 観光庁「インバウンド消費動向調査」

※いずれも各機関が公開しているオープンデータを利用しています。

## データの取得方法

データは以下の手順で取得・利用することを想定しています。

1. 各公式サイト（JNTO、観光庁）から公開統計データを取得  
2. 年度・四半期・国／地域単位で必要なデータを選択  
3. ローカル環境において「data/」 配下に配置  
4. Notebook および Streamlit アプリから参照

本リポジトリには、元データおよび加工済みデータのファイルは含めていません。

## 前処理および利用上の前提条件

取得した統計データは、分析および可視化を行う前段として、以下の前処理を行うことを前提としています。

- 複数年・複数ファイルに分かれたデータの結合
- 国・地域名、項目名称の表記ゆれの整理
- クロス集計形式データの整形（ロング形式への変換等）
- 数値項目（訪日客数、消費単価、実施率など）の型変換・欠損値処理

## 再配布について

本プロジェクトは分析設計および実装例を示すことを目的としており、データファイルそのものの再配布は行っていません。

データの利用にあたっては、各データ提供元が定める利用条件に従ってください。
